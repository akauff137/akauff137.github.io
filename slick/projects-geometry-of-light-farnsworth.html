Geometry of Light

Client: Luftwerk

Temporary Installation

This project is an adaptation of the Geometry of Light install in Barcelona, but this time, a little closer to home. We put the laser installation up in the Mies van der Rohe designed Farnsworth House in Plano, IL. This installation was larger in scale, as it included the surrounding grounds as well as the house. On the second pass, I was able to make improvements on the stability of the code, and I was able to be more flexible on site with programming changes.

Technical Design

The technical backend of this project is nearly identical to the installation in Barcelona, for this project, I only had to scale up the number of recievers and lasers. We also added a second, self contained network that ran a seperate routine from the main house. For more information on the technical elements of this project, look at Geometry of Light - Barcelona.
